# Oasis Infobyte Level 2 Task 3 - Login Authentication

A simple login and registration system with a secured dashboard page.

## Files
- `index.html`
- `style.css`
- `script.js`
- `README.md`

## How to run
1. Put all files in one folder.
2. Open `index.html` in a browser.
3. Register first, then login.

## Features
- Register and login forms
- Local storage based authentication
- Secured dashboard view
- Logout option
- Responsive modern UI
